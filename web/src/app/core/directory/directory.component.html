Bagaimana hendak ke Planetarium?

Menggunakan Kereta Api

Ambil kereta api dan lakukan perjalanan ke stesen MRT Muzium Negara (SBK15) melalui akses jejantas pejalan kaki.

Menggunakan Bas

Ambil bas dari KL Sentral ke Muzium Negara - 7 Minit jalan kaki ke Planetarium Negara.

Menggunakan Grab

Buat tempahan perjalanan e-hailing dengan Grab dengan memilih untuk ambil atau turun di Planetarium Negara.

Menggunakan Kereta

Mulakan perjalanan ke Planetarium Negara dari Jalan Bangsar terus menuju ke Stesen Kereta Api KTM memalui pusingan, belok ke kiri ke arah Masjid Negara.
Mulakan perjalanan ke Planetarium Negara dari Jalan Mahameru, ambil kiri ke Jalan Parlimen. Di lampu isyarat, belok kanan kea rah Taman Tasik Kuala Lumpur.
Atau anda boleh waze "Planetarium Negara".

Menggunakan Teksi

Memilih perjalanan ke Planetarium Negara dengan menggunakan teksi adalah lebih mudah dengan hanya meminta untuk menurunkan anda di hadapan Planetarium Negara. Sentiasa meminta pemandu teksi untuk hidupkan meter tambang sebelum memulakan perjalanan.

Menggunakan Motorsikal
